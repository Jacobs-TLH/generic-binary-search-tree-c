# Generic Binary Search Tree in C

A generic Binary Search Tree (BST) implementation written in C.

This project demonstrates how to build reusable data structures in C by converting an integer-specific binary search tree into a generic implementation capable of storing arbitrary data types through user-provided comparison and management functions.

## Features

* Generic Binary Search Tree implementation
* Dynamic memory management
* Insertion
* Search
* Deletion
* Tree traversal
* Modular design using separate header and source files

## Technologies

* C
* CMake
* Dynamic Memory Allocation
* Function Pointers
* Generic Programming

## Project Structure

* `BNode` - Binary tree node implementation
* `BSearchTree` - Binary Search Tree implementation
* `main` - Example usage and testing

## Build

```bash
mkdir build
cd build
cmake ..
make
```

## Run

```bash
./GenericBST
```
